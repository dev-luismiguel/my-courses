# Meat - Angular App Starter

## 1. Steps to start

### Dependencies install

`npm install`

### Initializing the server

`ng serve` or `npm start`

## 2. Backend

### json-server install

`npm install -g json-server`

### Initializing the database

`json-server db.json`


### To run with SSL:

`nodemon --watch backend/dist/server.js`

`node backend/dist/server`
