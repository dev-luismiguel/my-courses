# Meat - Angular App Starter

## 1. Steps to start

### Dependencies install

`npm install`

### Initializing the server

`ng serve` ou `npm start`

## 2. Backend

### json-server install

`npm install -g json-server`

### Initializing the database

`json-server db.json`
